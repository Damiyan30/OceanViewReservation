OceanViewReservation is a simple hotel reservation system. Use the menu to add reservations, view reservation details, and calculate/print bills.

Login

  • Go to login.html
  • Enter your username and password
  • After successful login, you will be redirected to the Dashboard

If you are not logged in, the system will redirect you back to the login page.

Add Reservation

  • Dashboard → Add Reservation
  • Fill required fields: Guest Name, Contact No, Check-in, Check-out, Room Type
  • Click Save Reservation
  • The system will show a Reservation ID (keep it to view/print bill)

View Reservation

  • Dashboard → View Reservation
  • Enter the Reservation ID
  • Click Search to view reservation details

Calculate / Print Bill

  • Dashboard → Calculate / Print Bill
  • Enter the Reservation ID
  • Click Load Bill
  • Click Print to open the browser print dialog

Bill is calculated using: Total = Nights × Nightly Rate

Logout (Exit)

  • Dashboard → Logout (Exit)
  • This ends your session and returns you to the login page

Troubleshooting

  • Unauthorized / redirected to login: your session expired or you logged out.
  • Reservation not found: check the Reservation ID is correct.
  • Invalid dates: check-out must be after check-in.
  • Room types not loading: ensure /api/room-types works and DB has room types.

Support

If the problem continues, contact the system administrator.
