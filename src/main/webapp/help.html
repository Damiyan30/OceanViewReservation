How to use the system

Common questions and quick steps.

FAQ

How do I login?

Open login.html, enter your credentials, then you’ll be redirected to the Dashboard.

How do I add a reservation?

Dashboard → Add Reservation → fill required fields → Save. You’ll get a Reservation ID.

How do I view a reservation?

Dashboard → View Reservation → enter Reservation ID → Search.

How do I print a bill?

Dashboard → Bill → enter Reservation ID → Load Bill → Print.

Why do I get “Unauthorized”?

Your session expired or you logged out. Login again.
